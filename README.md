## 🚀 Quick Start

### Prerequisites
- Python 3.11 or higher
- Git LFS for large model files
- Virtual environment (recommended)

### Installation

```bash
# Clone the repository
git clone https://github.com/JlovesYouGit/agent-97.git
cd agent-97

# Create virtual environment
python -m venv model_env
model_env\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Initialize models
python model_weight_installer.py
```

### Basic Usage

```python
# Initialize AGI components
from adaptive_formula_generator import AdaptiveFormulaGenerator
from model_weight_recalibrator import ModelWeightRecalibrator
from sha256_mathematical_validator import SHA256MathematicalValidator

# Create AGI instances
formula_generator = AdaptiveFormulaGenerator("0009095353")
weight_recalibrator = ModelWeightRecalibrator("0009095353")
math_validator = SHA256MathematicalValidator("0009095353")

# Generate adaptive formulas
result = await formula_generator.generate_adaptive_formula(
    "Create a formula for population growth with consciousness integration"
)

# Validate mathematical formulas
validation = await math_validator.validate_formula(
    result["generated_formula"]
)

print(f"Formula: {result['generated_formula']}")
print(f"Validation: {validation['is_valid']}")
```
